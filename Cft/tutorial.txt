Инструкция по использованию программы CFT

При запуске программы необходимо указать команды в следующем порядке:
1) java - запускает class файл.
2) -cp  -путь к папке где лежат скомпилированные файлы. 
3) Указать имя файла с функцией main
4) Указать аргументы для запуска (всего 4 аргумента ):

	1. Имя входного файла. Напрмер: "in.txt"
	2. Имя выходного файла. Например: "out.txt"
	3. Режим сортировки: 
		  по возрастанию: "a"
		  по убыванию   : "d"
	4. Тип содержимого: 
		  строки        : "s"
		  целые числа   : "i"
		  
Пример команды для целых чисел по возрастанию: 
java -cp ./out ru.atomskih.cft.Main in.txt out.txt i a 

Пример команды для целых чисел по убыванию: 
java -cp ./out ru.atomskih.cft.Main in.txt out.txt i d

Пример команды для строк по возрастанию: 
java -cp ./out ru.atomskih.cft.Main in.txt out.txt s a

Пример команды для строк по убыванию: 
java -cp ./out ru.atomskih.cft.Main in.txt out.txt s d
